# 卡牌每日養成 · 全自動戰鬥（Card Daily-Cultivation Auto-Battle）

一款純前端的卡牌養成放置型自動戰鬥網頁遊戲。每日領取資源、抽卡取得素材與稀有卡、
用素材升級角色、編排 5 人前後排陣容，觀看 ATB 速度條 + 能量條驅動的全自動戰鬥。

## 技術棧
- **Vite + 原生 JS（ES modules）**
- **PixiJS**：戰鬥畫面（目前以佔位圖形呈現：色塊／圓形／血條）
- **GSAP**：攻擊、受擊、大招等動畫特效
- **localStorage**：本機存檔

## 開發

```bash
npm install
npm run dev        # 開發伺服器（預設 http://localhost:5173）
npm run build      # 產出 dist/
npm run preview    # 預覽產出
npm run test       # 跑單元測試（vitest）
```

## 玩法
- **每日 12:00 發放**：跨過中午即可領取「抽卡券 + 金幣」（規則為佔位值，可於 `src/systems/daily.js` 調整）。
- **抽卡**：一抽一個。高機率養成素材、低機率稀有角色卡（**不分 R/SR/SSR**）。重複角色自動轉素材。
- **升級**：消耗養成精華 + 金幣提升角色等級，提高 HP/ATK/DEF/SPD。
- **陣容**：最多 5 人，可指定前 / 後排。
- **戰鬥（全自動）**：
  - **ATB 速度條**：依 SPD 累積，滿格即行動。
  - **能量條**：行動 / 受擊累積，滿格自動施放大招。
  - **普攻前排優先**：先攻擊敵方前排，前排全滅才打後排。
  - **5 屬性相剋**：火 > 風 > 水 > 火（循環），光 ↔ 暗（互剋）。剋制 ×1.5、被剋 ×0.75。
  - **職業**：坦克（高血防、減傷大招）／輸出（高攻、爆發大招）／輔助（治療大招）。
  - 掛機自動連戰，勝利推進關卡並獲得金幣。

## 架構（戰鬥引擎與渲染解耦）

```
src/
├── core/      狀態、存檔、事件、亂數、數值推導
├── data/      屬性相剋、職業、角色卡、素材、抽卡表（平衡常數集中於此）
├── systems/   抽卡、升級、陣容、每日發放、戰鬥佈署
├── battle/    純邏輯戰鬥引擎（ATB / 能量 / 選敵 / 傷害 / 技能）— 不依賴 DOM/Pixi，可單元測試
├── render/    PixiJS 場景 + GSAP 特效（訂閱引擎事件）
└── ui/        HUD / 分頁 / 抽卡 / 角色養成介面
```

**設計重點**：`src/battle/` 完全是純邏輯，靠事件對外溝通，因此可用 vitest 充分測試，
也方便日後替換美術或調整平衡，而不動到核心。所有平衡 / 機率 / 相剋常數集中在 `src/data/`。

## 測試
單元測試涵蓋：屬性相剋倍率、普攻前排優先、ATB 行動順序、能量滿放大招、
勝負判定、抽卡權重、升級扣資源、每日 12:00 邊界判定。

```bash
npm run test
```
